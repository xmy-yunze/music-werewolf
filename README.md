# 🎵 音乐狼人杀

一个用 Spring Boot 开发的多人音乐推理游戏。

## 🎮 游戏简介

音乐狼人杀是一款结合了音乐与社交推理的派对游戏。每位玩家会听到一首歌，其中有一名玩家（卧底）听到的歌与其他玩家不同。通过线下肢体语言，找出隐藏在人群中的卧底。

## ✨ 核心功能

- 创建/加入房间（支持4人）
- 20+ 首歌曲库，包含中文、英文、经典老歌
- 自动分配卧底，3首大众歌 + 1首卧底歌
- B站链接播放音乐，无需担心版权问题
- 单元测试自动验证游戏逻辑

## 🛠 技术栈

- Spring Boot 3.5.16
- Java 17
- Gradle
- JUnit 5

## 📡 API 接口

| 接口 | 方法 | 说明 |
|------|------|------|
| `/api/room/create` | POST | 创建房间 |
| `/api/room/join` | POST | 加入房间 |
| `/api/room/info/{roomId}` | GET | 查看房间信息 |
| `/api/room/start/{roomId}` | POST | 开始游戏 |
| `/api/room/bilibili/{roomId}/{playerName}` | GET | 获取B站歌曲链接 |
## 📋 游戏流程
房主创建房间，生成6位房间号
1. 3-4名玩家加入房间
2. 房主点击"开始游戏"
3. 系统自动分配歌曲（3首相同 + 1首不同）
4. 每位玩家获得自己的B站链接，前往听歌
5. 2分30秒后投票，找出卧底
