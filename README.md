# 🎵 音乐狼人杀

> 一款基于 Spring Boot 开发的多人音乐推理游戏 —— 听歌识曲，找出卧底！


## 🎮 游戏简介

音乐狼人杀是一款结合了**音乐**与**社交推理**的派对游戏。

每位玩家会随机分配到一首歌，其中有一名玩家（卧底）听到的歌曲与其他玩家不同。玩家需要在 **2分30秒** 内听完歌曲，然后通过讨论和投票，找出隐藏在人群中的卧底。


## ✨ 核心功能

| 功能 | 说明 |
|------|------|
| 🏠 房间管理 | 创建/加入房间，支持4人同玩 |
| 🎵 智能选歌 | 20+ 首歌曲库，自动分配 3首相同 + 1首不同 |
| 🕵️ 卧底机制 | 随机指定卧底，每局游戏身份不同 |
| 🔗 B站播放 | 点击链接跳转B站听歌，无需担心版权 |
| ⏱️ 倒计时 | 2分30秒自动进入投票阶段 |
| 🗳️ 投票系统 | 实时统计票数，自动公布结果 |
| 📱 移动适配 | 手机浏览器直接访问，完美适配 |
| 🧪 单元测试 | 一键验证游戏逻辑 |


## 🖥️ 前端页面展示

| 页面 | 路径 | 功能说明 |
|------|------|----------|
| 🏠 **首页** | `/` | 输入昵称，创建房间或加入房间 |
| 👥 **等待大厅** | `/lobby/{roomId}` | 显示玩家列表、房间号，房主点击开始游戏 |
| 🎵 **游戏页面** | `/game/{roomId}/{playerName}` | 显示歌曲信息、B站链接，2分30秒倒计时 |
| 🗳️ **投票页面** | `/vote/{roomId}/{playerName}` | 选择卧底，提交投票 |
| 📊 **结果页面** | `/result/{roomId}` | 公布卧底身份和投票结果 |

### 页面截图

#### 首页
- 输入昵称，一键创建房间或加入房间
- 自动生成6位房间号，方便分享

#### 等待大厅
- 实时显示已加入的玩家列表
- 房主可见「开始游戏」按钮
- 显示房间号，支持一键复制分享

#### 游戏页面
- 显示分配的歌曲名称和歌手
- 点击「去B站听歌」跳转播放
- 2分30秒倒计时，时间到自动进入投票

#### 投票页面
- 显示所有可投票的玩家
- 点击选择目标，确认投票

#### 结果页面
- 公布卧底身份
- 显示投票统计
- 提示游戏胜负

## 🚀 快速开始

### 环境要求

- JDK 17+
- 现代浏览器（Chrome / Safari / Edge）

### 克隆项目

```bash
git clone https://github.com/xmy-yunze/music-werewolf.git
cd music-werewolf
```

### 运行项目

```bash
./gradlew bootRun
```

### 运行测试

```bash
./gradlew test
```

### 访问游戏

**本地访问：** 打开浏览器访问 **http://localhost:8080**

**在线演示：** 打开浏览器访问 **http://121.41.167.211:8080**
